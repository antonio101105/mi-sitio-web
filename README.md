# ASIR_hub

Plataforma de recursos para el grado de Administración de Sistemas Informáticos en Red (ASIR).

## Estructura

El contenido está dividido por cursos:
- **1º Curso**: FH, DIG, GBD, LMSGI, SOS, IPE1, ISO, PAR.
- **2º Curso**: ASO, ASGBD, SAD, PROY, OPT, IPE2, IAW, HLC, ING, SRI.

## Tecnologías
- HTML5
- CSS3 (Glassmorphism)
- JavaScript (Vanilla)
